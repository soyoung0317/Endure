Q.	CPU와 GPU의 차이에 대해 설명하시오 
A.	컴퓨터의 연산을 수행하는 핵심 프로세서이지만, 용도와 구조가 다르다.

	1. 역할
		CPU는 "복잡한 연산을 빠르게", GPU는 "많은 연산을 동시에" 수행하는 것이 특징!

			구분			CPU (중앙처리장치)			GPU (그래픽처리장치)
		주요 역할	논리적 연산, 프로그램 실행		그래픽 및 병렬 연산
		작업 방식	직렬(Sequential) 연산			병렬(Parallel) 연산
		연산 속도	단일 코어 성능이 높음				다수의 코어로 동시 연산
		사용 목적	OS 운영, 앱 실행, AI 연산			그래픽 처리, 머신러닝, 병렬 계산
		구조			소수의 강력한 코어				다수의 약한 코어

	2. 차이점
		1) CPU
			소수의 고성능 코어 (4~16개)
			직렬(Sequential) 연산에 최적화
			논리 연산, 분기 예측, OS 운영, 일반 애플리케이션 실행
			= CPU는 강력한 단일 코어 성능을 제공하지만, 대량의 연산을 동시에 수행하는 데는 한계가 있음.

		2)  GPU의 구조
			수천 개의 작은 코어 (CUDA, Stream Processor 등)
			병렬(Parallel) 연산에 최적화
			그래픽 렌더링, AI 연산, 딥러닝, 대규모 행렬 연산 수행
			= GPU는 병렬 처리가 뛰어나므로, 다량의 데이터 처리 및 벡터 연산에 강함
